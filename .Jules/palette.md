## 2024-05-03 - Social Link Generic Components\n**Learning:** When using generic components that accept a `platform` prop for external links, simple `aria-label` interpolation (e.g., `Visit my ${platform} profile`) drastically improves screen reader context over just reading the platform name. Combined with `border-radius` and simple hover transitions, generic text links can feel like robust, delightful interactive elements without adding new UI dependencies.\n**Action:** Always add interpolated `aria-label` attributes and basic CSS transitions to generic repeating link components to ensure accessible context and interactive delight.
